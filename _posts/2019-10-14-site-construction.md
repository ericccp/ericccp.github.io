This site was built using code from [Github.dev](https://github.dev/). The site is powered by Jekyll and Github Pages. I have included helpful links below that be might useful if you want to create a project like this.

Links I found helpful:
 - [Primer CSS Documentation](https://primer.style/css/) - Primer CSS was used to style this site and having the documentation handy makes it easy to make customizations.
 - [Github.dev Tutorial](http://bit.ly/2VE733Y) - This tutorial shows you how to use the Github.dev site generator using Git and the command line. If this is your first project, make sure to add SSH keys to your Github account for easier access.
 - [Github SSH Keys Guide](http://bit.ly/33zjLn4) - This guide walks you through generating SSH keys on your local machine that you can add to your Github account allowing access to your repositories from the command line without using passwords.
